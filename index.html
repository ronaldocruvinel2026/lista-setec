Adicionar produtos de uma planilha

Selecione ou arraste o arquivo Excel (.xlsx). Os produtos serão somados à lista atual.

Formato da planilha (.xlsx): a primeira linha deve conter os títulos das colunas, na ordem: FABRICANTE, PRODUTO, PRINCIPIO ATIVO (I.A) e PREÇO. A coluna Princípio Ativo é opcional. Preços em branco ou zerados entram como "sem preço". Para separar por tipo, use abas com o nome DEFENSIVOS e/ou ESPECIALIDADES — se a planilha tiver só uma aba, tudo entra como Defensivos. Produtos novos são adicionados; produtos com fabricante + nome já existentes têm o preço atualizado automaticamente.
Exemplo de como a planilha deve estar:
FABRICANTEPRODUTOPRINCIPIO ATIVO (I.A)PREÇO
SYNGENTAELESTAL NEOAcetamiprido + Lambda245,80
NORTOXGLIFOSATO 480Glifosato32,50
